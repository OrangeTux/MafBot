Mafbot
======
A simple bot for the maffiagame www.maffiaworld.nl.

Installation
------------
Create a virtualenv and install all modules:

    virtualenv .env
    source .env
    pip -r install requirements.txt


Configuration
-------------
Set your ingame username and password in `mafbot/conf.py`

    username = 'example'
    password = '11235811'

Running
-------

    ./mafbot.py
