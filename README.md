MafBot
======
A simple bot for the maffiagame www.maffiaworld.nl

Installation
------------
Create a virtualenv and install the required packages:

    virtualenv .env
    source .env
    pip install -r requirements.txt

Confuguration
-------------
Set your credentials in `mafbot/conf.py`:

    username = 'user'
    password = '11235811'

Run
---

    ./mafbot.py
